MCP Servers Reference

Think of MCP servers as power tools that plug into Claude Code. The plugin bundles Context7, which gives Claude instant access to up-to-date documentation for WordPress, PHP, JavaScript, and 100+ other frameworks. Need to know how the Interactivity API handles state? Context7 fetches the answer in real-time instead of relying on outdated training data.

Known Issue: Auto-Loading

Sometimes MCP servers don't wake up automatically. If Claude can't look up docs, you'll need to add the server manually. See Manual Configuration for the fix.

Context7

Ever ask Claude about a framework and get an answer from outdated training data? Context7 fixes that. It's a documentation service that keeps Claude current with 100+ frameworks--WordPress, PHP, React, Next.js, whatever you're using. Think of it as having the official docs piped directly into Claude's brain.

Tools Provided

Tool Description
resolve-library-id Maps "WordPress" to the actual library identifier Context7 uses. You don't call this--Claude does it automatically
get-library-docs Fetches the actual documentation pages. Ask "How does the Interactivity API work?" and this grabs the latest WordPress docs

Manual Configuration

If the server doesn't load automatically (you'll know because Claude can't fetch docs), you need to wire it up yourself. It's a one-minute copy-paste job.

Project-Level Configuration

To enable for just this project, add this to .claude/settings.json in your project root:

{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}

Global Configuration

Or enable everywhere--every project on your machine gets this server. Add to ~/.claude/settings.json:

{
  "mcpServers": {
    "context7": {
      "type": "http",
      "url": "https://mcp.context7.com/mcp"
    }
  }
}

Requirements

Server Requirement
Context7 Internet connection (HTTP endpoint, no local install needed)

Verifying MCP Servers

After you add the config, restart Claude Code. Then test that everything works:

# Ask Claude what it has
"What MCP tools do you have access to?"

# Test Context7 (should fetch real docs)
"Look up WordPress Block Editor documentation"

# If it fails, double-check your JSON syntax and file paths
